Drupal Theme

While working at Weill Cornell, I built a responsive theme for our base Drupal distribution. It uses jQuery for DOM manipulation and was built using Sass to modualrize the styles. I wrote with the design team in mind- they will be building various sites based off this theme, and they have a theme file that includes the relevant styles that they can override easily, along with a set of color variables to make their jobs easier. One of the most interesting parts of this was the navigation, and making it functional across screen sizes.

The theme has been applied to Radation Oncology, Pain Medicine, and Anesthesiology
